# TRANSPFLOW - Blockchain-Based Supply Chain Management System

A decentralized supply chain management system built on Ethereum, ensuring transparency, security, and traceability in product tracking. Users can add, edit, remove, and view products securely, leveraging blockchain technology to maintain an immutable ledger of transactions.<br/>
<br/>
✨ Features<br/>
<strong>Decentralized & Secure</strong> – Uses Ethereum blockchain to prevent tampering and ensure data integrity.<br/>
<strong>Product Lifecycle Management</strong> – Users can add, update, and remove products, ensuring real-time tracking.<br/>
<strong>Immutable Recordkeeping</strong> – Every transaction is stored on the blockchain, providing a transparent and verifiable history.<br/>
<strong>Web3 Integration</strong> – Seamlessly connects with MetaMask and other Web3 wallets for blockchain interactions.<br/>
<br/>
🛠 Tech Stack<br/>
<strong>Frontend:</strong> React.js (with Vite), Tailwind CSS.<br/>
<strong>Blockchain:</strong> Ethereum (Smart Contracts in Solidity)<br/>
<strong>Web3 Library:</strong> Web3.js<br/>
<br/>
🔧 Prerequisites<br/>
1)Vite With TailwindCSS<br/>
2)Node.js (v16+ recommended)<br/>
3)MetaMask Wallet (Chrome extension)<br/>
4)Ganache (for local blockchain testing)<br/>
<br/>
📥 Installation Steps<br/>
1) Clone the Repository<br/>
```
git clone https://github.com/UTKARSHJHA-2005/TranspFlow.git
cd TranspFlow
```
<br/>
2) Install the required dependencies<br/>
``` 
npm install
```
<br/>
3) Run the server<br/>
```
npm run dev
```
